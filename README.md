## Skills

### code-golf

[`code-golf`](code-golf/SKILL.md) is a code quality mode for writing the shortest source code that fully solves the whole problem. It is meant for requests like "code golf this," "make it concise," "reduce boilerplate," "shortest diff," "minimal implementation," or "smallest change."

The name borrows from [code golf](https://en.wikipedia.org/wiki/Code_golf), which Wikipedia describes as a recreational programming competition where programmers try to produce the shortest source code that solves a problem. This skill applies that idea pragmatically: brevity is valuable only after behavior, edge cases, fallbacks, and error paths are preserved.
